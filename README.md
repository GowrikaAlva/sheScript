# SheScript – AI Prescription Translator for Women

SheScript is an AI-powered web application that converts complex medical prescriptions into **simple explanations designed for women**.
Users can enter or upload a prescription and instantly receive clear information about the medicine, dosage, safety warnings and usage instructions.

The goal of SheScript is to help women especially elderly patients, rural users and first-time mothers **understand their prescriptions and take medicines safely.**

---

## Features

- Converts medical prescriptions into **simple language explanations**
- Shows **medicine purpose and safe dosage**
- Highlights **pregnancy and hormonal safety warnings**
- Provides **food and drug interaction alerts**
- Generates a **daily dosage checklist**
- Supports **multiple languages**

---

## Tech Stack

- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **AI Processing:** Google Gemini API
- **Translation:** MyMemory Translation API
- **Database:** MongoDB
- **Backend:** Node.js + Express

---

## How It Works

1. User enters a medicine name or uploads a prescription image  
2. Gemini AI analyzes the prescription  
3. The system generates a simplified explanation  
4. Safety warnings and dosage instructions are displayed  
5. The result can be translated into the user’s language  

---

## Project Purpose

SheScript **does not diagnose diseases or replace doctors.**  
It simply helps patients **understand their prescriptions clearly and safely.**
